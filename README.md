# Mega-data - Application de Gestion des Membres d'Église

Application web complète pour la gestion des membres d'une église avec React, Node.js, Express, SQLite et JWT.

## Fonctionnalités

### Pour les membres :
- Inscription et connexion sécurisée
- Formulaire d'enregistrement des membres (nom, prénom, numéro, quartier)
- Visualisation des membres enregistrés
- Tableau de bord personnel

### Pour les administrateurs :
- Visualisation de tous les membres
- Gestion de tous les utilisateurs
- Statistiques complètes
- Suppression de membres
